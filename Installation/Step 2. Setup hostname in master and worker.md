In master node:

```
#Setup hostname:
sudo hostnamectl set-hostname "K8-master"

# To implement the change immediately in CLI, use the below command
exec bash 
# There you can see the hostname change to k8-master

# to confirm the hostname (one more check)
hostname

```

In Worker node: repeat the above command 

```
#Setup hostname:
sudo hostnamectl set-hostname "K8-worker"

# To implement the change immediately in CLI, use the below command
exec bash 
# There you can see the hostname change to k8-master

# to confirm the hostname (one more check)
hostname

```
